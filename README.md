## Mass Send Military Governor

A Simple QOL mod. This creates a end all military governors to join war interaction, instead of sending them one by one. Made this one for my own convenience.

Features:
- Interaction to send all military governors to aid with war
- Dynamic cost calculation based on number of governors sent.
